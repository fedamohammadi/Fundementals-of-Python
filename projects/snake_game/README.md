# 🐍 Snake Game (Python + Pygame)

This is a classic Snake game built using Python and Pygame as part of my learning and practice in programming and project development.

The goal of the game is simple: control the snake, eat the food, and grow as long as possible without hitting the walls or yourself. As the snake eats more food, the game gradually becomes faster, making it more challenging over time.

## 🚀 Features

- Smooth snake movement with keyboard controls  
- Score tracking  
- Increasing difficulty (speed increases as you eat food)  
- Improved snake visuals with a more natural look  
- Start screen with player name input  
- Game over screen with restart and exit options  

## 🎮 Controls

- Arrow keys → Move the snake  
- Enter → Submit name  
- Mouse → Click buttons (Start, Restart, Exit)  
- ESC → Exit the game
